# 💰 CapBase

Aplicación full stack para la gestión de finanzas personales.

## 🚀 Tecnologías

### Backend
- Java 17
- Spring Boot
- Spring Security
- JWT (autenticación)
- JPA / Hibernate
- MySQL

### Frontend
- React
- TypeScript
- Vite
- Axios

---

## 🔐 Funcionalidades

- Registro y login de usuarios
- Autenticación con JWT
- Protección de endpoints
- Gestión de movimientos (ingresos/gastos)
- Listado de movimientos del usuario autenticado
- Logout
